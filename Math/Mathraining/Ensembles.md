Ensemble : collection d'objets
Si un objet $a$ appartient à l'ensemble $A$, on le note $a\in A$ 

Définition par extension : citer explicitement tous les éléments d'un ensemble
Ex : $A=\{1;2;3\}$ 
Définition par compréhension : donner une propriété de tous les éléments de l'ensemble
Ex : $B=\{x\in\mathbb N|x\le5\}$ 

On se fiche de l'ordre d'apparition de chaque élément dans un ensemble, ou de leur nombre d’occurrence
Ex : $\{1;2;3\}=\{2;3;1\}=\{1;1;2;3\}$ 

Notions courantes : 
Un ensemble peut-être fini ou infini. Si $A$ est fini, sa taille est appelé son cardinal et se note $|A|$ ou $\#A$ 
Lorsque $A$ est inclus dans $B$ et a un cardinal strictement inférieur, on note $A\subset B$.
Lorsque deux ensembles sont les même, on note $A\subseteq B$ (attention, dans ces cas là : $A\not\subset B$)
L'union se note $A\cup B$ (Ex : $\{1;2\}\cup\{1;3\}=\{1;2;3\}$)
L'intersection se note $A\cap B$ (Ex : $\{1;2\}\cap\{1;3\}=\{1\}$)
La différence se note $A\setminus B$ (Ex : $\{1;2\}\setminus\{1;3\}=\{2\}$)
Le complémentaire d'un ensemble $A$ est l'ensemble de tous les éléments appartenant à l'univers $U$, mais pas a $A$ et se note $A^c$
Ex : $U=\{1;2;3\}, \{1\}^c=\{2;3\}$ 

Ensembles usuelles : 
$\emptyset$ est l'ensemble vide
$\mathbb N$ est l'ensemble des nombres naturels, donc entiers et positifs
$\mathbb Z$ est l'ensemble des entiers relatifs, donc positifs et négatifs
$\mathbb Q$ est l'ensemble des rationnels
$\mathbb R$ est l'ensemble des réels
$\mathbb C$ est l'ensemble des nombres complexes
Nous avons : $\emptyset\subset\mathbb N\subset\mathbb Z\subset\mathbb Q\subset\mathbb R\subset\mathbb C$ 

Les $n$-uplets : 
Un groupe de $n$ éléments dans lequel l'ordre a de l'importance (Ex : $(1,2)$ est un $2$-uplet)
On peut définir les types des éléments dans un $n$-uplet avec $\times$, qui est le produit cartésien
Ex : $(\pi,\frac{1}{2},-1)$ fait partie l'ensemble $\mathbb R\times\mathbb Q\times\{-1;-2\}$
